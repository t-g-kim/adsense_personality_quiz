# GitHub Pages 배포 가이드

## 배포 방법

1. GitHub 저장소 접속:
   https://github.com/t-g-kim/adsense_personality_quiz

2. 상단 "Settings" 탭 클릭

3. 왼쪽 메뉴에서 "Pages" 클릭

4. "Source" 섹션에서:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)

5. "Save" 버튼 클릭

6. 1-2분 대기

7. 페이지 새로고침 → 배포 URL 확인

## 배포 URL
https://t-g-kim.github.io/adsense_personality_quiz/

## 커스텀 도메인 (선택사항)
Settings → Pages → Custom domain에서 설정 가능
